# Rudraksh Bhardwaj — Portfolio

Minimal text-first portfolio built with Next.js 15, TypeScript, and Tailwind CSS.

## Getting started

```bash
npm install
npm run dev
```

## Before deploying

1. Add assets to `public/`:
   - `avatar.png`
   - `favicon.ico`
   - `resume.pdf` (optional)

2. Update content in `src/data/`:
   - `site.ts` — bio, writing link, social URLs
   - `experience.ts`, `projects.ts`, `education.ts`

## Build

```bash
npm run build
```

Static files export to `out/`.
